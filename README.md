## 🔍 Troubleshooting common issues

If you encounter difficulties, review these common solutions.

*   The trainer does not detect the game: Ensure that the game is running before you open the trainer. Refresh the process list if the button exists in the tool.
*   Security warnings: Windows might flag external tools as unrecognized. This happens because the tool reads game memory to apply changes. Add an exclusion in your security software if the program will not run.
*   Changes do not apply: Verify that you are using the version of the trainer that matches your current version of the game. Reload your save file if the trainer remains active but the game state does not update.
*   Game crashes: Close both the game and the trainer. Reopen the game first, load your save, and then reopen the trainer. Avoid activating multiple high-impact features at the same time if the game becomes unstable.
